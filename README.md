# Modura Analytics Kazi Pro

Build recruiter-ready resumes, cover letters, and job emails in minutes.

## Project Description

Modura Analytics Kazi Pro is a free, browser-based career tool designed to help job seekers create professional, ATS-friendly resumes, cover letters, and job application emails. The tool uses rule-based logic to generate clean, simple documents that pass through Applicant Tracking Systems (ATS) without requiring any AI, API keys, login, or payment.

## Features

### Resume Builder
- Form fields for personal details, skills, experience, education, projects, and certifications
- Generates ATS-friendly resume with simple formatting (no tables, columns, or images)
- Copy to clipboard, download as PDF, download as Word
- Auto-saves form data to localStorage

### Cover Letter Builder
- Form fields for personal info, target role, company, skills, and job description
- Generates professional, recruiter-friendly cover letters
- Tailors content to match job description keywords
- Copy, PDF, and Word download options

### Job Application Email Builder
- Creates professional job application emails with subject lines
- Includes value proposition and contact information
- Copy to clipboard functionality

### ATS Score Checker
- Evaluates resume against 8 ATS compliance criteria
- Displays score out of 100 with color-coded feedback
- Provides specific tips for improvement
- Categories: Strong (80-100), Good (60-79), Weak (Below 60)

### Additional Features
- 100% private — all data stays in your browser
- No server, no login, no data collection
- Works offline
- Mobile responsive design
- localStorage persistence — your data survives page refreshes

## Technologies Used

- **HTML5** — Page structure and forms
- **CSS3** — Professional styling, responsive design, print styles
- **JavaScript (ES6+)** — Form handling, document generation, localStorage, Blob API

## How to Run

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Start building your resume, cover letter, or email

No installation, no dependencies, no build tools required.
